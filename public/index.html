🌤️ About GET WEATHER

GET WEATHER is a simple, clean weather app that gives you the current weather conditions of any city in the world — including temperature, humidity, description, and a live map of the location.

✨ Features

  • 🌡️ Live temperature in °C
  • 💧 Humidity and weather description
  • 🌥️ Weather emoji for quick glance
  • 🗺️ Interactive map using Leaflet
  • 🎨 Dynamic background based on temperature

🔧 Built With

HTML, CSS, JavaScript, OpenWeatherMap API, Leaflet.js, OpenStreetMap

📜 Credits

Weather data: OpenWeatherMap
